# WeatherConditionApp
## Overview
This project is a Python-based weather application developed for the COMP1002 - Advanced Python course. It aims to real-time weather data for a user-specified city using an external API and displays the current temperature, humidity, and general weather conditions. The goal of this project is to apply advanced Python concepts, work with RESTful APIs, and process JSON data in a practical setting.

## Features
* Retrieves real-time weather data from ["cfd93bd25c18114a13f4dd3c8f7f531f"].
* Parses and displays key metrics such as temperature, humidity, and weather description.
* Handles invalid city inputs with error handling
* Provides customized daily activity recommendations based on the current weather conditions using a dictionary-based mapping
* Includes clickable UI elements that allow users to seamlessly navigate and view weather forecasts for upcoming days, extending the functionality beyond just the current weather.
